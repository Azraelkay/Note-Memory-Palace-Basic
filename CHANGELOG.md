# 更新日志

所有重要的项目更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-01-04

### 新增
- 🎨 记忆宫殿主题设计
- 📝 富文本笔记编辑器
- 🗂️ 多级分类管理系统
- 🔍 全文搜索功能
- 👤 用户注册/登录系统
- 🔐 JWT身份验证
- 💾 自动保存功能
- 📱 响应式设计
- 🖥️ Electron桌面应用支持
- 🎭 多主题切换（记忆宫殿、星空、深渊）
- ✨ 启动动画效果
- 🏰 自定义应用图标

### 技术特性
- React 18 前端框架
- Flask 后端API
- SQLite 数据库
- Context API 状态管理
- CSS3 动画效果
- 模块化组件设计

### 界面特色
- 宇宙蓝背景配色 (#0B1426)
- 金色神经网络元素 (#FCD34D)
- 紫色渐变装饰 (#7C3AED)
- 流畅的过渡动画
- 现代化卡片设计
